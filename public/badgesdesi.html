[Untuk Dosen]
  • Selama masa awal perkuliahan, aktivitas IMPORT konten perkuliahan dilakukan pembatasan pada pukul 07.00-19.00 WIB.
  • Panduan Penambahan Simulasi Interaktif PHET ke dalam kelas untuk mendukung kuliah atau praktikum.
  • Panduan IMPORT MATERI untuk penggandaan konten kelas.
  • Untuk melihat kelas dari semester sebelumnya di dashboard. Pada Course Overview, ubah filtering menjadi "Past"
  • Lisensi ZOOM akan diberikan pada akun Bapak/Ibu Dosen jika melakukan start meeting melalui myITS Classroom. Demi kelancaran , mohon Bapak/Ibu Dosen melakukan 3 (tiga) langkah berikut:
    • 1. Selalu membuat (create meeting) melalui myITS Classroom di halaman course masing-masing.
    2. Selalu memulai (start meeting) melalui myITS Classroom dengan menekan tombol Start Meeting berwarna hijau di halaman course masing-masing. Mohon untuk tidak memulai (start meeting) langsung dari aplikasi Zoom desktop.
    3. Tombol start meeting sudah tersedia 20 menit sebelum waktu terjadwal dimulainya meeting. Bapak Ibu sangat disarankan memulai meeting lebih awal agar terhindar dari kepadatan akses, kemudian pelaksanaan kelas (meeting)-nya tetap dimulai sesuai jadwal perkuliahan.

  • Untuk melihat kelas dari semester sebelumnya di dashboard. Pada Course Overview, ubah filtering menjadi "Past"
